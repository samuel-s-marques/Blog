### Local installation
1. Install [Node.js](https://nodejs.org/en/)
2. Install [MongoDB](https://www.mongodb.com/)
3. ```git clone https://github.com/samuel-s-marques/Blog.git```
4. ```cd Blog```
5. ```npm install```
