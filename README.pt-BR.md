### Instalação local
1. Instale [Node.js](https://nodejs.org/pt-br/)
2. Instale [MongoDB](https://www.mongodb.com/pt-br)
3. ```git clone https://github.com/samuel-s-marques/Blog.git```
4. ```cd Blog```
5. ```npm install```
